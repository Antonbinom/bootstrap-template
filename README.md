# BlueRex

## О проекте

Pet-проэкт. Одностраничный сайт сверстан с использованием Bootstrap.
<br>Так же сайт был интегрирован в CMS Wordpress

## Функционал

- Адаптивная и валидная верстка
- Прелоадер
- Табы
- Слайдер
- Галерея

## Стэк технологий

- Bootstrap
- JQuery
- Wordpress
